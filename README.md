# STOOR - Jie-Jun Yeh

Front-end for [fakestoreAPI](https://fakestoreapi.com/docs).

## Instructions

Install dependencies by running `npm i`, then run `npm run dev` to start the local server.

To run tests, use `npm run test`.

## Tech Stack

- Typescript
- React
- Material UI
- Vite
